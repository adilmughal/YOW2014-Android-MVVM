YOW Connected 2014 - MVVM Implementation in Android.

This repository contains sample projects related to talk "Write cleaner, maintainable and testable code in Android using MVVM" delivered by Adil Mughal at YOW Connected 2014.

The projects demonstrate two ways of implementing MVVM in Android

  • RoboBinding
  • Java PropertyChangeSupport/Listeners
